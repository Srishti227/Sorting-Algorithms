## Bubble Sort
Bubble sort is a simple sorting algorithm

### Algorithm
1. Compare adjacent elements.
2. Swap if they are in wrong order
3. Repeat until sorted.

### Time Complexity
- Best: O(n)
- Average: O(n^2)
- Worst: O(n^2)

### Selection sort
Selection sort is a simple comparision based sorting algorithm.
It repeatedly selects the smallest element from the unsorted portion of the array and places it at the beginning
