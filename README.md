# Bubble-sort
Bubble sort implementation in C with versioned releases

## Bubble Sort
Bubble sort is a simple sorting algorithm

### Algorithm
1. Compare adjacent elements.
2. Swap if they are in wrong order
3. Repeat until sorted.

### Time Complexity
- Best: O(n)
- Average: O(n^2)
- Worst: O(n^2)
